Design like a PRO by adding a layout grid to your Elementor page while designing.

== Description ==

A super simple plugin to add a layout grid to your page when designing in Elementor.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress

== Frequently Asked Questions ==

= How do I use Elementor Layout Grid? =

With the Elementor editor open, click the cog icon on the lower left hand side of the editor to open the page settings.

= Do I need Elementor PRO with this add-on? =

No, you don't. It'll work with both free version of Elementor and Elementor PRO.
